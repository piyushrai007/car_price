car price prediction

"In this project, a predictive model for estimating car prices has been developed using machine learning techniques. The dataset includes various features such as the car's year of manufacture, selling price, present price, kilometers driven, fuel type, seller type, transmission type, and the number of owners. The initial data analysis involved exploring unique values, checking for missing values, and creating a correlation matrix to understand relationships between variables. The dataset was preprocessed by creating a new feature, 'year_old,' representing the age of the car. Categorical variables were one-hot encoded, and the final dataset was split into features (X) and target variable (y). An Extra Trees Regressor was trained on the data to determine feature importances. Subsequently, a Random Forest Regressor model was optimized using Randomized Search Cross-Validation, considering various hyperparameters. The final model achieved good predictive performance, as demonstrated by a scatter plot comparing actual and predicted selling prices. The model has been saved for future use as 'random_forest_car_price.pkl.' This project demonstrates the application of machine learning in predicting car prices, providing valuable insights for both buyers and sellers in the automotive market."
